About Stir Craft

A community-driven cocktail catalog for recipes, collections, and inspiration.

What is Stir Craft?

Stir Craft is a lightweight recipe sharing app focused on cocktails. Users can create, edit, and share cocktail recipes, build curated lists, and discover drinks by ingredient, vessel, or vibe.

Features

  • Create and manage cocktail recipes with ingredient formsets
  • Build lists and favorite recipes
  • Search and filter by ingredient, vessel, and color
  • Quick-add and AJAX-powered list management for smooth UX

Documentation

See the project docs for setup and contribution guidelines, or browse the technical guides in the docs folder.

The Team

Built by the Stir Craft team — passionate about cocktails, UX, and clean code. Contributions welcome via pull requests.
